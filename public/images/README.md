# Rock N Soul Suite — Image Assets

Drop Facebook images in this folder. The site references these filenames (update the `page.tsx` file if you use different names):

- `gig-1.jpg` — Live music shot (used in Hero and What's On sections)
- `gig-2.jpg` — Crowd / venue shot
- `interior.jpg` — Bar interior
- `bar-shot.jpg` — Close-up of drinks / bar detail
- `band.jpg` — Band performing

If a file is missing the site will fall back to stock placeholders — but the real thing is always better.
